# debugger-pod

Projekt debugger-pod je vytvořen jako jednoduchý nástroj pro debugging HTTP hlaviček v prostředí Kubernetes. Jedná se o Go aplikaci, která spouští HTTP server na portu 8080. Server přijímá příchozí HTTP požadavky, extrahuje z nich všechny hlavičky a vrací je klientovi ve formátu JSON.
